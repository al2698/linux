# 3-1

 CentOS 關閉 selinux

```
# getenforce //檢查 SELinux 的狀態

gedit /etc/selinux/config //永久關閉 SELinux
SELINUX=disabled
```

 查看防火牆狀態命令：

```
sudo firewall-cmd --state //可查看當前防火牆的狀態
sudo systemctl disable firewalld //重新開機後永久關閉
```

ping IPv6：

```
ping -6 2001:b400:e757:3dc5:cdfd:d051:8826:4da2
```
